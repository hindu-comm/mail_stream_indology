This is easily done using TeX or LaTeX together with the Devanagari font
and pre-processor by Charles Wikner.  See the INDOLOGY file archives
(via http://www.ucl.ac.uk/~ucgadkw/indology.html) and the Comprehensive
TeX Archive Network (CTAN) at ftp://ftp.tex.ac.uk under
tex-archive/languages
