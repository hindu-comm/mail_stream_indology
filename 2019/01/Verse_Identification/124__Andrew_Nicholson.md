Dear Colleagues,

The following verse appears in Madhva's and Baladeva's commentaries on the
Brahmasūtra. I suspect it appears in one or more Purāṇas:

puṇyapāpādikaṃ viṣṇuḥ kārayet pūrvakarmaṇā /
anāditvāt karmaṇaś ca na virodhaḥ kathañcana //

Please do let me know where you found this if you have run across this
śloka in a smṛti text somewhere.

All the best,
Andrew
