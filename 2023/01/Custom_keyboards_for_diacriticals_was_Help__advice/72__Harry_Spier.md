Dear list members,
Custom keyboards can be created for the Mac assigning any unicode character
or combination of characters to any key using the application Ukalele from
SIL https://software.sil.org/ukelele/
The instructions come with the download and its very straightforward to use
(even for this Windows user :-) ).  I've sent a solution to Madhav with the
diacritics he needs.

For Windows users  its even more straightforward. In microsoft word,
documents called templates can be created that re-assign keys to any
unicode character you want. These templates are then attached to any
document you want to use the new keyboard with.


Harry Spier
