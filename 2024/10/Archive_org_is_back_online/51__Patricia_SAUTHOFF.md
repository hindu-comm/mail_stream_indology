Dear all,

The Internet Archive has posted a blog about the current availability of its services. https://blog.archive.org/2024/10/21/internet-archive-services-update-2024-10-21/

It is "read-only", which means those beautiful pdfs we usually see aren't showing up on the main page. If you scroll down and look to the right, you will see "Download Options". If you click pdf there you should be able to get a downloadable version. The site is running slowly.

Currently, users are also unable to log in, so any pages you have favorited won't be available. This hack did also include a data breach, so I would recommend, even if you don't use it often, to change your password. This function does seem to be available, though without being able to fully log in it's not totally clear.

We all likely use the Wayback Machine less than the archive books feature, but that fortunately does seem to be up and running as before and is a wonderful resource.

Happy online researching!
