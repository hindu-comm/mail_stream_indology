Respected Scholars,

I had been bringing some Sanskrit lexica in digitized format earlier.
In the same sprint, I did digitize some commentarial literature on Sanskrit
kośas.
I have been trying to get them proof-read, but it has been taking way too
long.
Therefore, I have decided to put them on the list 'as is'.
I would appreciate the correction submissions, if the list members come
across errors.
There are 20 digitized resources.
https://github.com/sanskrit-kosha/kosha is the github repository hosting
all digitized lexical resources posted on the lists so far.
I apologize for the cross-posting, but think that the list members may be
interested in the information.

With regards,
Dhaval
