I made a few tests in this regard: With Firefox and Chromium on 
different Linux installations from an Indian ip address, I got the error 
message. It did work, however, with Tails OS (also a Linux system) and 
the Tor browser (that changes the geolocation all the time).

It may help if you download the command line version of Pancanga, latest 
version is 3.14. This is a perl script and can be invoked in a terminal 
window on Linux or MacOS (and other Unix-a-like systems) with

perl pancanga3.14
