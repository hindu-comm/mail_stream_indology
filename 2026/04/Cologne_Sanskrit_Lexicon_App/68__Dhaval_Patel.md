Dear Friends,

Majority of us are already using Cologne Sanskrit Lexicon project
dictionaries available at https://www.sanskrit-lexicon.uni-koeln.de.

It gives me immense pleasure to announce the following apps which allow
offline and faster access to the latest Cologne data. You can start typing
the word in your desired transliteration scheme and the results will be
shown on the fly. Data would be fetched from latest data from Github
repositories where Cologne d

Kindly give it a try and give your valuable feedback.

*Web* - https://sanskrit-lexicon.github.io/csl-app/

*iOS / macOS* -
https://apps.apple.com/in/app/cologne-sanskrit-lexicon/id6760785804

*Android* -
https://play.google.com/apps/testing/de.uni_koeln.sanskrit_lexicon (Testing
version - may require you to send email to me to add you to testers.
Testers are required to use the app for 14 consecutive days as per Google
Play Console terms, so that it may be published as an app on Play Store.)

*Web App* - Some browsers like Chrome may allow you to install you the web
app directly on your computer from
https://sanskrit-lexicon.github.io/csl-app/

*Usage Instructions -*

1. Click on the triple bar on the left top corner.
2. Download the dictionaries you want to use from 'Manage Dictionaries'
3. Click on the 'Back' arrow to go back to the Home Page.
4. If you want to search 'Headwords', type in the first search bar.
5. If you want to search inside 'Definitions', type in the second search
bar.
6. If you want to search for the entries having given word in 'Headword'
and some other given word in 'Definition', type in both search bars.

By default, the searches will be treated as 'prefix' searches.
You can change them to 'exact', 'suffix' or 'substring' searches from
'Preferences'.
You can also change your preferred input and output transliteration schemes
from ‘Preferences’.

If you find an error or would like to verify the printed text, you can
click on 'PDF' or 'Correction' links below each entry.

Note - Users using the web or web app version should not download too many
dictionaries on the first go. Data resides in your browser cache. Once you
are confident that your computer and browser are capable of handling
installed dictionaries, gradually scale up. I tested with all dictionaries
in Safari on MacOS with 16GB RAM. It was working well, but it may differ
from machine to machine.


Dr. Dhaval Patel, I.A.S.
